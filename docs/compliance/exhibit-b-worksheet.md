# NDPA Exhibit B Worksheet — pre-filled for DailyWins

*Drafted 2026-06-11. Exhibit B is the data-elements checklist in the NDPA the
district will hand us. This worksheet is our answer key so whoever fills the
real form checks ONLY these boxes. When in doubt, the answer is "No."*

## Check YES

| Exhibit B category | Element | Our note |
|---|---|---|
| Application Use Statistics | Meta data on user interaction | Staff sign-in timestamps; per-period entry counts |
| Schedule | Student scheduled courses / periods | Period numbers only (Period 0–8); no course titles per student |
| Special Indicator (only if district uses notes this way) | — | Teacher free-text notes may reference goals; we store no coded indicators. Discuss with LEA; default **No**. |
| Student In App Performance | Program/application performance | The behavior marks themselves (the product's purpose) |
| Student Name | First and/or Last | Teacher-entered display name |
| Other | Teacher-authored observational notes | Shared/private flag; private = staff + official record only |
| Other | Student self-assessments (optional feature) | Stored separately from official record |

## Check NO (everything else, explicitly including)

Student identifiers (SSN, state ID — *optional local student ID field may be
added for placement reporting; if/when enabled, amend Exhibit B*), date of
birth, demographics (race, ethnicity, gender, ELL, FRPL), enrollment history,
grades/transcripts, assessment scores, attendance as system-of-record
(per-period absent markers exist only as scoring context), health records,
IEP/504 documents, discipline records, address/phone/email, parent/guardian
identifiers (parents access via tokenized links — we never collect parent
names or emails), photos, video, audio, biometric data, geolocation, device
identifiers, social media, transportation, food service.

## Statements that go with the form

- Purpose of collection: classroom behavior-goal tracking and progress
  reporting, configured and operated by the LEA's (or NPS's) own staff.
- Data is collected **by school staff**, not from students directly (the
  optional self-assessment is the single exception, teacher-enabled).
- No data is used for advertising, profiling, or product training; no sale or
  rental ever (SOPIPA).
